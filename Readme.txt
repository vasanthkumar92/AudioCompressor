AUDIO COMPRESSOR Readme file.

AudioCompression:
1.The Algorithm wrapper is in block processing method
2.The block size is 256

Steps To Run the Algorithm:
1.Open the AudioCompressor_Wrapper.m file
2.Set the input audio file name
3.Set the parameter values in the drc_Param Strucuture.
4.Run the Algorithm
5.The output wav file will be generated.
